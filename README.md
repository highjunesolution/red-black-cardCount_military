# 🃏 Card Counter App

แอปนับใบแดง/ดำ สำหรับติดตามจำนวนใบที่เหลืออยู่แบบ Real-time พร้อมบันทึกประวัติและเปอร์เซ็นต์

---

## 📋 ภาพรวม

Card Counter App ช่วยให้ผู้ใช้สามารถ:
- ตั้งค่าจำนวนใบแดงและดำเริ่มต้น
- กดปุ่มเพื่อนับใบที่ถูกเปิดออก
- ดูเปอร์เซ็นต์ใบที่เหลือแบบ Real-time
- ดูประวัติการเปิดพร้อมเวลา (เรียงจากล่าสุดขึ้นก่อน)
- ลบรายการประวัติและคืนค่าจำนวนใบ
- บันทึกข้อมูลอัตโนมัติผ่าน `localStorage`

---

|---|---|

## 🚀 วิธีใช้งาน

### 1. เริ่มต้น
เมื่อเปิดแอปครั้งแรก จะมี Modal ให้ระบุจำนวนใบ:
- **ใบแดง** — จำนวนใบแดงเริ่มต้น
- **ใบดำ** — จำนวนใบดำเริ่มต้น
- กด **บันทึก** เพื่อเริ่มใช้งาน

> หากเคยใช้งานแล้ว แอปจะโหลดข้อมูลเดิมจาก localStorage อัตโนมัติ

### 2. นับ
- กดปุ่ม 🔴 เพื่อบันทึกว่าเปิดใบแดง 1 ใบ
- กดปุ่ม ⚫ เพื่อบันทึกว่าเปิดใบดำ 1 ใบ
- จำนวนที่เหลือและเปอร์เซ็นต์จะอัปเดตทันที
- ปุ่มจะถูก Disable อัตโนมัติเมื่อจำนวนใบเป็น 0

### 3. ดูประวัติ
ประวัติการจับจะแสดงทางด้านขวา เรียงจาก **ล่าสุดขึ้นก่อน** พร้อมเวลา (HH:MM:SS)

### 4. ลบรายการ
กดปุ่ม **ลบ** หน้ารายการใด รายการนั้นจะถูกลบและจำนวนใบนั้นจะถูกคืนค่า 1 ใบ

## 📁 โครงสร้างโค้ด

```
project/
├── index.html       # โครงสร้าง HTML + Bootstrap Modal
└── script.js        # Logic ทั้งหมด
```

### ฟังก์ชันหลักใน `script.js`

| ฟังก์ชัน / Event | หน้าที่ |
|---|---|
| `DOMContentLoaded` | โหลดข้อมูลจาก localStorage หรือแสดง Modal |
| `saveBtn` click | บันทึกค่าเริ่มต้นและซ่อน Modal |
| `redBtn` click | ลด redCount, บันทึกประวัติ |
| `blackBtn` click | ลด blackCount, บันทึกประวัติ |
| `updatePercent()` | คำนวณและแสดงเปอร์เซ็นต์ |
| `showList()` | Render รายการประวัติ (เรียงใหม่→เก่า) |
| `list` click (removeBtn) | ลบรายการ และคืนค่าจำนวนใบ |

## 🐛 Bugs ที่แก้ไขแล้ว

| # | บัค | สาเหตุ | วิธีแก้ |
|---|---|---|---|
| 1 | `blackCount` โหลดค่าผิด | `getItem("redCount")` ถูกใช้สำหรับทั้งคู่ | เปลี่ยนเป็น `getItem("blackCount")` |
| 2 | `blackCount` ไม่ถูกบันทึก | `setItem("redCount", blackCount)` ผิด key | เปลี่ยนเป็น `setItem("blackCount", blackCount)` |
| 3 | `listData` มี `null` ปนอยู่ | `listData.push(JSON.parse(...))` push null เมื่อไม่มีข้อมูล | ตรวจสอบก่อน push และใช้ spread `...parsed` |
| 4 | ปุ่มลบทำงานผิดรายการ | `splice(index, 1)` ได้รับ index เป็น string จาก `dataset` | แปลงด้วย `parseInt(index)` |
| 5 | count ไม่อัปเดตใน localStorage เมื่อกดปุ่ม | ขาด `setItem` หลังกด redBtn/blackBtn | เพิ่ม `localStorage.setItem` ในทุก click handler |
| 6 | ปุ่มไม่ถูก disable เมื่อโหลดข้อมูลเก่า | ไม่มีการตรวจสอบเมื่อ `DOMContentLoaded` | เพิ่ม `if (redCount === 0) redBtn.disabled = true` |

## 📝 หมายเหตุ

- ข้อมูลจะถูกบันทึกในเครื่องผ่าน `localStorage` ไม่มีการส่งข้อมูลออก
- หากต้องการเริ่มนับใหม่ ให้ล้าง localStorage หรือเพิ่มปุ่ม Reset ในอนาคต
